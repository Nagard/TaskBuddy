# TaskBuddy

TaskBuddy ist eine einfache Aufgabenverwaltung, die direkt im Browser läuft.
Die Daten werden im Local Storage gespeichert.

## Features (Sprint 1)

1. Aufgaben hinzufügen
2. Aufgaben anzeigen
3. Aufgaben als erledigt markieren
4. Aufgaben löschen

## Technologie-Stack

- HTML, CSS, Vanilla JavaScript
- GitHub Pages für Deployment
- GitHub Actions für CI/CD

## Wie starten?

1. Repo klonen
2. `index.html` im Browser öffnen
3. Aufgaben hinzufügen, abhaken oder löschen – Daten bleiben im Browser gespeichert
